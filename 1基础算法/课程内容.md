# 快速排序
** 思路：基于分治思想  **

1. 确定分界点（随机取任意一个数为分界点，一般取中点）；
2. 调整区间，把小于x的数移到左边，把大于x的数移到右边，把区间分为[l, j]、[j + 1, r]；
3. 递归左右。


## 快速排序算法模板
```
void quick_sort(int q[], int l, int r)
{
    if (l >= r) return;

    int i = l - 1, j = r + 1, x = q[l + r >> 1];
    while (i < j)
    {
        do i ++ ; while (q[i] < x);
        do j -- ; while (q[j] > x);
        if (i < j) swap(q[i], q[j]);
    }
    quick_sort(q, l, j), quick_sort(q, j + 1, r);
}
```
